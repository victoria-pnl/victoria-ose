# Victoria Coach PNL — Programme OSE

Landing page du programme d'accompagnement **OSE** de **Victoria Puttemans**, coach en Programmation Neuro-Linguistique (PNL).

---

## Le projet

Ce site présente le programme OSE, un accompagnement de 90 jours destiné aux femmes débordées qui souhaitent sortir de la procrastination et retrouver de la clarté pour avancer à leur rythme — sans pression, sans perfection.

## Le programme OSE

**Durée :** 12 semaines (90 jours)  
**Prix :** 300 €  

**Ce qui est inclus :**
- 3 appels personnalisés (début, milieu, fin)
- 1 livret PDF par semaine
- Suivi WhatsApp pendant 3 mois
- Défis d'action hebdomadaires

Un **livret découverte gratuit** est également proposé pour comprendre pourquoi on procrastine, identifier son profil de blocage et se remettre en mouvement en 3 jours.

## Victoria Puttemans

Coach certifiée en PNL, Victoria accompagne les femmes dans leur développement personnel en utilisant des outils doux et bienveillants. Elle propose aussi des séances de coaching individuelles en cabinet (Grez-Doiceau, Hélécine) et en ligne.

## Développement

Built with Next.js 16, React 19, Tailwind CSS v4 et Framer Motion.

```bash
npm install
npm run dev   # http://localhost:3000
npm run build
```

Pour modifier le prix, les liens Stripe, Calendly ou WhatsApp → `src/config/site.ts`
